# Fused Gaming - Organization Repository

Welcome to the Fused Gaming organization repository! This repository serves as the central hub for coordinating projects, managing organizational documentation, and maintaining governance policies for the Fused Gaming community.

## Repository Structure

```
.
├── .github/              # GitHub-specific configuration
│   ├── ISSUE_TEMPLATE/  # Issue templates for different types of requests
│   ├── profile/         # Organization profile README
│   ├── workflows/       # GitHub Actions workflows
│   └── *.md            # Governance, contributing, and other docs
├── .vscode/             # VS Code workspace configuration
├── .editorconfig        # Editor configuration
└── .prettierrc.json     # Code formatting configuration
```

## Purpose

This repository manages:

- **Project Coordination**: Templates and workflows for organizing projects
- **Issue Templates**: Standardized forms for bugs, features, goals, and governance proposals
- **Documentation**: Guidelines for contributing, governance, security policies
- **Community Engagement**: Interactive elements like the tic-tac-toe game
- **Organization Profile**: Public-facing information about Fused Gaming

## Getting Started

### Prerequisites

- VS Code (recommended)
- Git

### Local Setup

1. Clone this repository
2. Open the workspace file: `Fused Gaming.code-workspace`
3. Install recommended extensions when prompted
4. You're ready to contribute!

### Recommended VS Code Extensions

The workspace includes recommendations for:
- Markdown editing and preview
- YAML validation
- GitHub integration
- Git management (GitLens)
- Spell checking
- Code formatting
